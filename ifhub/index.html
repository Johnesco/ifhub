IF Hub

An Interactive Fiction Player & Source Browser

IF Hub is a browser-based player for Inform 7 games. Pick a game, play it online, and read the source code side by side — all in one interface. Every game runs entirely in your browser via Parchment, a JavaScript interpreter for the Glulx and Z-machine virtual machines. No downloads, no plugins.

The source viewer gives you syntax-highlighted Inform 7 code with a navigable outline, search, and line numbers. Games with walkthroughs include annotated guides. Some games feature ambient audio and sound effects that play alongside the text.

About

IF Hub is a static site — no server-side logic, no accounts, no tracking. It serves games built with Inform 7, a programming language that reads like natural English and compiles to virtual machine formats that have been around since the 1980s. The same technology that powered Infocom’s games now runs in any modern browser.

All source code is visible. The games, the player, and this site are open. If you want to see how any of it works, open the source tab while you play.

Built with Inform 7. Playable via Parchment.
