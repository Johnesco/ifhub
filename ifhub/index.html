IF Hub

An Interactive Fiction Player & Source Browser

IF Hub plays text adventures in the browser. Pick a game, play it online, and read the source code side by side — all in one interface. No downloads, no plugins.

Games

How It Works

Every game runs entirely in your browser. Parchment is a JavaScript interpreter that executes the same virtual machine formats that powered Infocom’s games in the 1980s. The games here span three eras of interactive fiction:

ZIL → Z-machine

Infocom’s original language. Zork I v0 plays the unmodified 1980s source, compiled with ZILF to a .z3 binary and run via Parchment’s Z-machine interpreter.

BASIC → Inform 7

Dracula’s Castle started as a type-in BASIC listing from the early 1980s. The original source is preserved with annotations; the playable version is a faithful translation to Inform 7.

Inform 7 → Glulx

Most games here are written in Inform 7, a language that reads like English prose. Source compiles to Glulx and runs via Parchment’s WASM engine.

Sound → Blorb

Games with audio embed .ogg files directly in the game binary as a Blorb package. Parchment plays them through the Web Audio API — no external files.

Source Viewers

Every game has a source browser. The Inform 7 viewer provides syntax highlighting, a navigable outline from Part/Chapter/Section headings, and search. Zork I v0 has a dedicated ZIL source browser covering all ten original Infocom source files with annotations explaining the game engine. Dracula includes both the original annotated BASIC listing and the Inform 7 translation side by side.

Testing & Toolchain

The games are tested with a deterministic walkthrough system. Each game has a walkthrough — a sequence of commands that completes the game. A golden RNG seed ensures combat, NPC behavior, and other random events produce the same outcome every run. The walkthrough runner pipes commands through native CLI interpreters (glulxe for Glulx, frotz for Z-machine) built from source, captures the transcript, and checks the final score.

The entire toolchain — compilation, blorb packaging, web player generation, testing, and deployment — runs from the command line. There is no IDE dependency. Inform 7 source compiles to Inform 6, then to a Glulx binary, optionally packaged with sound into a Blorb, base64-encoded into a JavaScript wrapper, and served as a static page. Each game is its own GitHub Pages site; the hub iframes them directly.

About

IF Hub is a static site — no server, no accounts, no tracking. All source code is visible. The games, the player, the toolchain, and this site are open. If you want to see how any of it works, open the source tab while you play.
